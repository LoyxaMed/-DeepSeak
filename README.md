## 🖥️ Демо

Проект доступен для локального запуска. В браузере откроется интерактивная панель с:

- Анимированным звёздным фоном ✨
- Живой телеметрией (скорость, высота, топливо, температура, давление, радиация) 📊
- Графиком активности, обновляемым в реальном времени 📈
- Мини-игрой «Найди звезду» с системой прогресса 🎯
- Треком орбитальной станции с движущимся кораблём 🗺️

---

## 🌟 Особенности

| Функция | Описание |
|---------|----------|
| **Звёздный фон** | 280 мерцающих звёзд с динамической яркостью. Рендерится в отдельном Canvas. |
| **Телеметрия** | 6 параметров обновляются каждые 1.8 секунды с реалистичными случайными значениями. |
| **График** | Линейный график с градиентной заливкой, отображает последние 45 точек данных. |
| **Мини-игра** | Игровое поле 5×5. Нужно найти 10 звёзд. После каждого клика появляются новые звёзды. |
| **Орбитальная карта** | Земля с вращающимся спутником на эллиптической орбите. Анимация плавная, 60 FPS. |
| **Адаптивность** | Корректно отображается на мобильных устройствах и планшетах. |
| **Производительность** | Все анимации используют `requestAnimationFrame`. Нет утечек памяти. |

## 🛠️ Технологии

Проект написан на **чистом веб-стеке** без фреймворков:

- **HTML5** — семантическая разметка с использованием современных тегов.
- **CSS3** — флексбоксы, гриды, градиенты, псевдоэлементы, бэкдроп-фильтры (стеклянный эффект), медиа-запросы.
- **JavaScript (ES6)** — модульная архитектура, работа с Canvas, обработка событий, таймеры, замыкания.
